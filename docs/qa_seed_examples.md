# Примеры для ручного QA (RU)

Ожидания: ответы **на русском**, без токсичности; картинки — по описанию, стиль **понятный неспециалисту**. Точный пиксель-матч не требуется.

## Consumer (10)

1. **Вопрос:** «Как правильно хранить суши дома до вечера?» — короткий бытовой совет, без медицинских обещаний.
2. **Вопрос:** «Что подарить папе на 55, если у него всё есть?» — идеи списком, без навязчивых ссылок.
3. **Картинка:** «Рыжий кот в шарфе, снег за окном, уютный стиль иллюстрации» — один кот, читаемый силуэт.
4. **Картинка:** «Торт с надписью «С юбилеем», пастельные тона» — надпись читаема.
5. **Поздравление:** «Сестре на 30 лет, весёлый тон, без пошлости» — 4–8 предложений + тёплое окончание.
6. **Поздравление:** «Коллеге с уходом на пенсию, уважительно и светло» — без жаргона HR.
7. **Шаблон ДР** (после выбора): отправить черновик с заменой «мама» → «дядя» — текст должен отразить замену.
8. **Шаблон открытка с цветами:** отправить черновик как есть — картинка с цветочной темой, светлый фон.
9. **Пустое сообщение** в режиме вопроса — `QUESTION_EMPTY` или аналог, без падения.
10. **Реферал:** неверный код — `unknown_code`; верный — `accepted`.

## Business (10)

1. **VK:** «Салон красоты, -15% на стрижку до 15 марта, адрес и телефон вымышленные» — структура: оффер, срок, контакт, призыв.
2. **VK:** «Кофейня, новый сезонный напиток, дружелюбно» — короткий пост, без лишней «воды».
3. **Картинка:** «Баннер распродажи обуви, крупный текст -20%, оранжевый акцент» — читаемый текст на баннере (если модель рисует текст).
4. **Картинка:** «Логотип для цветочного магазина «Лаванда», минимализм» — лаконичный знак/надпись.
5. **Шаблон распродажа:** черновик из меню — пост с выгодой и сроком.
6. **Шаблон новинки:** акцент на товар/поступление, призыв зайти.
7. **Шаблон услуга:** описание услуги + доверие (опыт, запись).
8. **Шаблон праздничная акция:** сезонность + мягкий призыв.
9. **Короткий VK:** шаблон «короткое объявление» — ≤ ~900 символов по смыслу, ёмко.
10. **Consumer в business-режиме** (если переключились): главное меню business-only действий; VK недоступен без тарифа — paywall `PAYWALL_VK_NOT_ENTITLED`.
